# Fase I — laboratorio

## Función

Esta carpeta absorbe el laboratorio mínimo reproducible de la Release 2.

## Archivos adscritos en este bloque

- `celula_custodia_sv_release2.py`
- `testigo_minimo_custodia_sv_release2.svp`
- `pseudocodigo_custodia_release2.txt`
- `checklist_sprint_release2.md`
- `salida_casos_canonicos_release2.json`

## Regla de lectura

Estos archivos no son accesorios decorativos. Son parte de la primera materialización visible de la custodia basal del diseño.

Su función es permitir contraste reproducible de:

- compuertas conservadoras;
- conservación de `U`;
- evaluación de la arquitectura compuesta;
- y trazabilidad de salida.
