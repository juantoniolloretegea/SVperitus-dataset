# Inyección A — endurecimiento operativo del AE-GD2-SV

Este lote endurece el agente en cinco frentes materiales:

1. secuencia soberana `S0…Sn` explícita y exportable;
2. banco doble: secuencia soberana y banco técnico `append-only`;
3. panel interactivo SV–Usuario enriquecido con métricas diferenciales, modales y metrológicas auxiliares;
4. bundle ZIP más rico, con CSV, JSONL, índices y prosa de auditoría;
5. referencia Python ampliada con los mismos bloques auxiliares.

No cierra todavía el corrector avanzado sobre imagen raster arbitraria, pero deja preparado el carril matemático y de auditoría para la iteración siguiente.
