# Matriz mínima de ensayo manual por navegador

Ensayar, como mínimo, en:

- Chrome estable actual y versión inmediatamente anterior
- Microsoft Edge estable actual y versión inmediatamente anterior
- Firefox estable actual y versión inmediatamente anterior
- Opera estable actual y versión inmediatamente anterior

Casos mínimos:

1. carga local de PNG/JPEG/SVG;
2. rechazo visible de SVG hostil;
3. construcción de `UserContextPacket`, `GraphicIntentDraft` y `GraphicIntentConfirmed`;
4. secuencia soberana `S0…S4`;
5. polígonos `C_frame^25` y `C_gob^25`;
6. métricas diferenciales, modales y metrológicas;
7. descarga del ZIP completo.
